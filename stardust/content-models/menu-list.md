# Menu List Content Model

## DA Table Structure

| Column 1 | Column 2 |
|-----------|----------|
| Item name + description + optional tag | Price |

### Column 1: Item Details
- **First line (bold or h3):** Item name
- **Second line:** Description text
- **Third line (optional, italic):** Tag like "Classic", "Spicy", "New"

### Column 2: Price
- Plain text price like "$18" or "$12 / $44"
